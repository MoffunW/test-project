# test-project

## [demo on firebase](https://test-project-mkc.web.app/#/).

## Установка пакетов

```sh
npm install
```

## Запуск

### С замоканами данными

```sh
npm run dev:mock
```

### С apiUrl
внутри `.env.development` в переменную `VITE_API_URL` подставить адрес

затем

```sh
npm run dev
```
